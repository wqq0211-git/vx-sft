# 第二轮训练方案

推荐目标：
- 在第一轮基础上补充更稳定、更贴近日常聊天的样本
- 控制回复长度，减少过度书面化
- 提高一致性，而不是一味增加花样

建议策略：
- 学习率低于第一轮
- 数据量少于第一轮也可以
- 优先修正明显 badcase
- 训练后做批量推理抽检

建议流程：
1. 准备 `outputs/round2_train.jsonl`
2. 准备 `outputs/round2_valid.jsonl`
3. 执行 `bash training/run_train_round2_ddp.sh`
4. 训练完成后执行 LoRA merge
5. 用 UI 做人工对话验证

# 双卡训练与进度查看

## 启动双卡训练

```bash
bash training/run_train_round2_ddp.sh
```

默认行为：
- 自动切到项目根目录
- 如果存在 `.venv` 会自动激活
- 默认使用 `CUDA_VISIBLE_DEVICES=0,1`
- 日志写入 `logs/`

## 查看进度

```bash
bash training/check_progress.sh
```

输出包括：
- 当前日志文件路径
- 训练进程 PID
- 最近日志尾部
- GPU 利用率

## 自定义 GPU

```bash
CUDA_VISIBLE_DEVICES=3,4 bash training/run_train_round2_ddp.sh
```

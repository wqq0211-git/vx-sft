# Contributing

感谢你对 `vx sft` 的关注。

## 贡献建议

欢迎提交以下内容：
- 文档改进
- 训练脚本优化
- 推理稳定性优化
- UI 体验优化
- 与隐私保护相关的改进

## 提交前建议

请尽量保证：
- 不提交任何私有聊天数据
- 不提交任何训练权重
- 不提交 `outputs/`、`checkpoints/`、`merged/` 等目录内容
- 改动尽量小而清晰
- 在 README 或 docs 中补充必要说明

## 推荐流程

1. Fork 仓库
2. 新建分支
3. 完成修改
4. 自查文档与命令是否可用
5. 发起 Pull Request

## 关于数据与隐私

这是一个强调隐私边界的模板仓库。
如果你的改动涉及示例数据，请只使用脱敏样例或自造样例。
